# Seem

Seem é uma plataforma de streaming fictícia desenvolvida como projeto front-end

## Páginas

- Landing Page — apresentação da plataforma
- Login & Cadastro — telas de autenticação
- Catálogo — listagem de filmes e séries disponíveis
- Pesquisa — simulação de resultados por algoritmo de recomendação
- Detalhes — página individual de um título
